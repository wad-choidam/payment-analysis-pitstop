# 결제 불일치 분석 도구

POS(BPOS/CPOS) ↔ Eximbay 단말기 간 결제 불일치 발생 시, 포스 로그와 단말기 로그를 입력하면 자동으로 파싱/분석하여 타임라인 흐름도와 상세 테이블로 시각화하는 웹 도구입니다.

## 실행

```bash
pnpm install
pnpm dev
```

## 테스트

```bash
pnpm vitest run
```

## 사용법

1. 포스 로그를 좌측 입력창에 입력 (`.xlsx` 엑셀 파일 드래그앤드롭 또는 텍스트 붙여넣기)
2. 단말기 로그를 우측 입력창에 붙여넣기 (선택 — 없으면 포스 로그만으로 분석)
3. "분석 시작" 버튼 클릭
4. 결론(중복결제 여부), 분석 요약, 결제 시도 분석, 타임라인, 상세 테이블 확인
